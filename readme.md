# Autonomy — a Logseq theme

Autonomy is a calm, human-centred theme for Logseq, designed around warm tones, expressive typography, and soft contrast.  
It draws inspiration from the Degrowth movement — simplicity, sufficiency, and focus over excess.

---

## Install

Add this line to your `custom.css` in Logseq:

```css
@import url("https://raw.githack.com/larsrose/autonomy-theme/main/theme.css?v=0.4.0");
